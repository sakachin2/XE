Introductions (sakachin2 projects)

  • Editor （Hybrid Editor XE) -->How-To-Use v1.30C

This editor is fit for programmers working on both main-frame and PC.
Its operation is based on mainframe SPF editor, and like as other editor on PC,
convenient short-cut key operation is hybrid.
Basic filer operation is also provided such as Delete/Copy/Move/Print.
It can also manipulate binary file.(Dump format or 3 line virtical Hex display mode)
Powerfull data file edit function is provided to analyze logfile etc by Excel.
Each version offer common operation on the differrent OS, DOS,OS/2,Windows, Linux and AIX.
Windows,Linux,AIX version support remote access by ftp/rsh.
Using s3270+IND$FILE+TSO,Win/Linux version support to access TSO file.
Win/Linux version support open by application such as excel,explorer
/OpenOffice,Nortilus using filename extension association.

アプリの紹介 (sakachin2 プロジェクト)

  • エディター (Hybrid Editor XE)-->取説 v1.30C

メインフレーム(SPFファイルの編集にも便利) と PC と 両方仕事している人に使い易いエディター。
ホストのSPFをベースにPCのEDITORの便利なキー操作をハイブリッド。
バイナリーエディター（ダンプ形式と３行平行表示形式と２種類サポート）,ファイラーとしても使えます。
LogfileなどをExcelで解析できるように編集する機能も強力。
各版がDOS, OS/2, Windows, Linux, AIX で共通の操作を提供。
Windows, Linux, AIX版ではftp/rshによるリモートアクセスサポート。
Windows, LinuxX版ではS3270+IND$FILE+TSOを利用してTSOファイルアクセスをサポート
WIN版/Linux版は拡張子による関連付けを利用した、
excel,explorer/OpenOffice,Nortilus等との連携もサポート

次のコマンドラインツールも同梱
  • xprint
    テキストファイル印刷
  • xads
    Windows NTFS の ADS (Alternative Data Stream)をアクセスする。
  • xbc
    簡易四則計算。ダブルワード（８バイト）精度、内部タイムスタンプ値の変換もサポート
  • xcv
    コード変換(EUC,SJIS,JIS,Unicode,UTF-8,EBCDIC,HexDigit)
  • xdc
    ディレクトリーの比較
  • xdd
    ディレクトリーの掃除
  • xdig
    ディレクトリーの階層を追ってコマンドを適用する。
  • xds
    ディレクトリーのサイズ表示
  • xfc
    ファイル内容の差異表示
  • xff
    ファイル名、タイムスタンプによるファイル探索.
  • xfg
    文字列を含む（バイナリー）ファイルの探索
  • xfmt
    テキストファイルのフィールドの桁あわせ（整列）、抽出
  • xfs
    巨大ファイルの分割統合
  • xkp (Windows版のみ)
    プロセスを表示、停止(kill)
  • xmj
    ２つのファイルの行照合、連結
  • xpe
    APIで設定されるerrno,GetLastError()の値の説明を表示
  • xts
    ファイルのタイムスタンプを設定
