=ScreenHardCopy

        HCOpy [EOL | NOEOL]

            Specifies whether lines are cut at the End-of-Line(EOL) or at the screen width (NOEOL).
            Effective until the end of the session.
            You can also specify this option by the command line option /Yh (EOL) (default is /Nh).
            This option is intended for the case lines in UTF8 files contain RightToLeft formatting characters.
