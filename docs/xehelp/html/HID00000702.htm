    = # 行コマンド
        .[n][,b][.s]はサポートされません
        .#のあとに以下のオプションが使用できます。
         ^ : コマンドを分割画面の他方の画面上で実行する。
         ! : コマンドがエラーで終了しても次の行のコマンドを実行する。

        .例。56行から58行までを分割画面の別の画面で実行する、各行のコマンドのエラーは
         無視して最後まで実行する。
         開いているファイルは::cl(コマンドログ)でタイムスタンプフィールドは
         無視される。(1桁目が"="のとき次の"="までを無視する)

            0055|s file1
            ##^ |i word1 -m all
            0057|i word2 -m all
            ##! |i wodr3 -m all
            0059|s
