    = DOS コマンド

#ifdef LNX
        [ DOS | > ] [コマンド文字列 | sh  [terminal-options] ] ]
#else
        [ DOS | > ] [コマンド文字列 | CMD ]
#endif

            DOS,OS/2 コマンドをコマンド入力行から入力できます
            コマンド文字列を指定しない場合 子シェルを開始します(EXITで
            終了)
            ファイルの経路名に *\ , **\ , ^*\ , :: が指定できます
            コマンドのパラメータとしてファイルの経路名としたくない場合は
            ２重引用符で囲んでください。
